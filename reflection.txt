Difficulties Faced
------------------
1. Understanding embeddings from scratch:
   We had no prior experience with sentence embeddings. We read the SentenceTransformers
   documentation and a few short articles to understand that embeddings are dense vector
   representations of text that capture semantic meaning. Once we understood the concept,
   the actual implementation turned out to be just a few lines of code.

2. Choosing a sentiment model that supports NEUTRAL:
   The default HuggingFace sentiment-analysis pipeline only returns POSITIVE or NEGATIVE.
   We needed a three-class model to also detect NEUTRAL sentiment as shown in the expected
   output. We found the cardiffnlp/twitter-roberta-base-sentiment-latest model which
   supports all three classes and resolved this issue.

3. Windows long path installation error:
   Installing PyTorch on Windows failed due to the 260-character path length limit.
   We worked around this by creating a Python virtual environment at a short path (C:\p1)
   and installing the CPU-only version of PyTorch, which avoids the long CUDA header files
   that caused the error.
